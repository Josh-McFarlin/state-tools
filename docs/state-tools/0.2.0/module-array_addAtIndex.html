array/addAtIndex

Add one or more item(s) to array at index.

Source:
  • array/addAtIndex.js, line 1
Since:
  • 0.1.3
Examples
addAtIndex([1, 2, 3], 1, 5)
    // => [1, 5, 2, 3]
addAtIndex([1, 2, 3], 1, 5, 6, 7)
    // => [1, 5, 6, 7, 2, 3]
