util/preserveState

An optional wrapper to preserve the original state if a state-tools function fails.

Source:
  • util/preserveState.js, line 1
Since:
  • 0.1.3
Examples
preserveState(removeIndex, [1, 2, 3, 4, 5], 1)
    // => [1, 3, 4, 5]
preserveState(removeIndex, [1, 2, 3, 4, 5], 'willError')
    // => [1, 2, 3, 4, 5]
